# ICOCA History Extractor

ICOCA（スマートICOCA・モバイルICOCA）の利用履歴ページからデータを抽出し、扱いやすい形式（CSV/JSON等）で保存するためのツールです。

## 概要

JR西日本が提供する「[ICOCA利用履歴照会](https://icoca.jr-odekake.net/pc/mbicocausehistory.do)」ページにログインし、表示される利用明細（日付、利用区分、運賃、残高など）を自動で取得します。

> **Note**: 本ツールはJ-WESTネット会員のログイン情報を利用します。公式のAPIではないため、サイトの仕様変更により動作しなくなる可能性があります。

-----

## 主な機能

  * **自動ログイン**: 指定したJ-WEST IDとパスワードを使用して履歴ページへアクセス。
  * **データ抽出**: ページ上のテーブルから「月日」「利用種別」「利用区間」「支払額」「残高」を抽出。
  * **フォーマット変換**: 抽出したデータをCSVまたはJSON形式で出力。
  * **期間指定**: 直近分だけでなく、表示可能な過去ログの取得をサポート。

-----

## 準備するもの

1.  **J-WESTネット会員登録**: [公式サイト](https://www.jr-odekake.net/cjw/about/)でICOCAを紐付けたアカウントが必要です。
2.  **実行環境**: Python 3.8以上
3.  **依存ライブラリ**:
    ```bash
    pip install selenium webdriver-manager pandas
    ```

-----

## 使い方

### 1\. 設定

環境変数、または `config.yaml` などの設定ファイルにログイン情報を記述します（セキュリティのため、コードへの直接記述は避けてください）。

### 2\. 実行

```bash
python extract_history.py --output history.csv
```

### 3\. 出力例

| 月日 | 利用種別 | 入場駅/事業者 | 出場駅/種別 | 支払額 | 残高 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 03/15 | 運賃 | 大阪 | 京都 | 580 | 2,420 |
| 03/14 | 物販 | コンビニ | --- | 210 | 3,000 |
